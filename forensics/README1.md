🕵️‍♂️ Hack4Gov Forensics Tryout Challenges

Welcome to the Forensics Category Tryouts for Hack4Gov.

This repository contains a series of digital forensic challenges designed to evaluate participants' analytical thinking, technical skills, and investigative methodology.

These challenges simulate real-world scenarios involving:

Network traffic analysis

File carving and embedded data detection

Data encoding and hex interpretation

Steganography and hidden information recovery

🎯 Objective

Participants are expected to:

Analyze provided evidence files

Identify suspicious artifacts

Extract hidden or encoded information

Document findings clearly and concisely

This is not only a test of tools, but a test of:

Attention to detail

Research ability

Logical reasoning

Proper forensic workflow

📂 Challenge Structure

Each folder contains:

📄 A short scenario description

📁 Evidence file(s)

❓ Guide questions

🏁 Flag format for submission

🏳 Flag Format

All flags follow this format:

IMPORTANT!!!! : FLAG{example_text_here}

Submissions must follow the exact format, including capitalization.

⚠ Rules

You may use any tools available to you.

Online tools are allowed.

Document your findings.

Do not brute-force or modify evidence files unless instructed.

Think before you act.

💡 Final Note

Some challenges are designed as warm-ups.
Others may require deeper investigation.

Good luck — and investigate carefully.
