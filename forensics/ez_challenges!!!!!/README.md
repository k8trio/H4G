# 🟢 Warmup Challenges – 5 POINTS EACH 

Welcome to the warmup section of the Forensics Tryouts.

These challenges are designed to test basic observation skills and encoding awareness.  
They are intentionally simple and meant to help you get comfortable before moving to more advanced investigations.

---

## 🎯 Objectives

Participants are expected to:

- Carefully examine provided files
- Identify obvious artifacts
- Recognize common encoding formats
- Pay attention to file details

---

## 🧩 Challenges Included

### 1️⃣ Obvious Strings - filename: f+controls_logs.txt
A simple file containing a handful of trash information.  
Sometimes the answer is right in front of you.

---

### 2️⃣ Hidden File Extension - filename: flag.jpg

A file may not always be what it appears to be.  
Investigate the file type carefully.

---

### 3️⃣ Metadata Never Lies - filename: meta_yarn.jpg

Images and documents often contain more than what you see on the surface.
Metadata can reveal hidden clues left behind by creators — sometimes intentionally.

Examine the file carefully. The answer may be stored in information most people forget exists.

---
### 4️⃣ Simple Encoding - filename: base64_reverse.txt

Encoded data is everywhere in cybersecurity. Recognizing common encoding formats is a fundamental skill for any analyst.

The file contains encoded text. Decode it properly and investigate the result carefully.

---
### 5️⃣ Hidden in Plain Sight - filename: hidden_in_plain_sight.docx

Not all hidden information is deeply buried or encrypted. Sometimes, the answer is placed right in front of you — just not in a way that’s immediately obvious.

Carefully inspect the provided file. Look beyond what is immediately visible and examine it from different angles. A small detail may reveal more than expected.
